# 🏪 מערכת ניהול חנות

פרויקט זה הוא מערכת לניהול חנות, הכוללת ניהול **מלאי מוצרים**, **עובדים**, ו**תשלומים**.  
המערכת נבנתה באמצעות **Angular** ומאפשרת ניהול יעיל ונוח של כלל פעילויות העסק.

---

## 📋 תכונות עיקריות

- 🔹 **ניהול מלאי מוצרים** — הוספה, עריכה ומחיקה של מוצרים, מעקב אחר כמויות ומחירים.  
- 🔹 **ניהול עובדים** — יצירת פרופילים לעובדים, תפקידים והרשאות.  
- 🔹 **ניהול תשלומים** — רישום עסקאות, תשלומים ללקוחות ולספקים.  
- 🔹 **ממשק משתמש נוח** — מבוסס על Angular Material / Bootstrap (בהתאם למה שנבחר).  
- 🔹 **API צד שרת** (אופציונלי) לתקשורת עם בסיס נתונים.

---

## 🛠 טכנולוגיות

- **Frontend:** Angular 17+  
- **שפת תכנות:** TypeScript  
- **ספריות עזר:** RxJS, Angular Material / Bootstrap  
- **ניהול גרסאות:** Git  
- **שרת פיתוח:** Angular CLI  

---

## ⚙️ התקנה והרצה

### דרישות מקדימות
- Node.js גרסה 18 ומעלה  
- npm או yarn  
- Angular CLI מותקן גלובלית:
  ```bash
  npm install -g @angular/cli
  ```

### הורדת הפרויקט
```bash
git clone https://github.com/username/store-management-angular.git
cd store-management-angular
```

### התקנת חבילות
```bash
npm install
```

### הרצת הפרויקט בסביבת פיתוח
```bash
ng serve
```
